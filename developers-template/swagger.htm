Formulas

GET​/v2​/formulas
Retrieves the collection of Formulas resources.
POST​/v2​/formulas​/bmi_calculator
bmi : body mass indicator = weight(kg) / height (m)^2
POST​/v2​/formulas​/cardio_calculator
Karvonen formula: % HR = (HRcurrent - HRrest) / (HRmax - HRrest)
POST​/v2​/formulas​/distance_speed_calculator
calculate sum of distance, and speed avg
POST​/v2​/formulas​/elevation_calculator
calculate ascent and descent with smoothing
POST​/v2​/formulas​/hr_kcal_calculator
Formula with HR (Kalenji2 A, source VALGEO report)
POST​/v2​/formulas​/hr_max_calculator
Maximum heart rate according to age and gender
POST​/v2​/formulas​/met_kcal_calculator
Formula with MET
POST​/v2​/formulas​/on_points_calculator
On points (man) = calorie x 100 / ( 0,0726 x 30 x weight ) On points (woman) = calorie x 100 / ( 0,0626 x 30 x weight )
POST​/v2​/formulas​/running_cooper_calculator
calculate MAS & VO2max with Cooper test based on the best performance during 12minutes
POST​/v2​/formulas​/running_record_calculator
calculate time in function of %MAS per distance
POST​/v2​/formulas​/weight_calculator
Ideal weight with the Lorentz formula in function of the gender
GET​/v2​/formulas​/{id}
Retrieves a Formulas resource.
